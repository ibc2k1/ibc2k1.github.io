about Run On

簡介:

Run On為韓國JTBC於2020年12月16日起播出的水木連續劇，此劇講述雖然使用著同樣的韓語卻難以溝通的時代，
生活在不同世界的人們用各自的語言溝通，各自以不同的速度向彼此靠近的戀情。

主要演員:

任時完 飾演 奇善謙

申世景 飾演 吳薇朱

崔秀英 飾演 徐丹雅

姜泰伍 飾演 李映禾

隨機台詞:

「希望細心又溫柔的人能夠過得很好，希望那些和善的人不要被當成笨蛋。」
